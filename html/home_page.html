How It Works

1. Create or Join a Tree

Create a private tree or join one with a code. Once you're inside, the tree, search, and profiles stay focused on that family.

2. View Your Family Tree

On the "Family Tree" page, you'll see everyone in the current tree organized by generation, with spouses and siblings grouped together.

Go to Family Tree

3. Add a Family Member

On the Family Tree page, use the + button in the bottom-right corner to add someone new to the current tree.

  • Required: first and last name.
  • Optional: birthday, middle initial, parents, and spouse.
  • Tip: add parents first so you can select them from the list.

4. View & Edit Profiles

Click on any person's card in the tree to open their profile. You'll see:

  • Birth date
  • Parents and children
  • Spouse information
  • A historical fun fact based on their birthday

From the profile page, you can update details after live testing confirms edits are working in your Firebase project.

5. Search for Family Members

Use the "Search" page to quickly find family members by name within the currently selected family tree.
