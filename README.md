# Family Tree App

A Firebase-powered family tree application for building, sharing, and browsing a private family tree. The current birthday release focuses on a polished signed-out demo, a signed-in owner flow, profile editing, photo support, and a large-tree chart that stays usable as the family grows.

## Overview

The Colety Family Tree App is a modern web tool for visualizing and managing family relationships. Owners can add relatives, edit profiles, upload photos, copy or reset invite codes, and use the Family Tree page to find, focus, and explore people.

The system uses Firebase Auth, Firestore, Firebase Storage, Firebase Hosting, and modular JavaScript to generate the tree layout, profile pages, account page, and CRUD flows. The public demo uses made-up example data; private Colety data should only appear to signed-in authorized accounts.

## Key Features

### Interactive Family Tree

Automatically arranges parents, partners, children, siblings, and extended branches into a chart-style family map with pan, zoom, find-person, selected-person details, and a card fallback.

### Add, Edit, And Remove People

Authorized owners/editors can add, edit, and remove people. Viewers and signed-out users stay read-only.

### Dynamic Profile Pages

Each person has a profile page generated by URL parameters and live Firestore lookups, with edit mode and photo handling for authorized users.

### Modular JavaScript Structure

`helpers.js`, `tree.js`, `familyChartSpike.js`, and the page modules keep the logic organized without introducing a build step.

### Birthday Prep Tools

Account and tree tooling helps prepare the birthday demo: invite code actions, missing-info checks, photo coverage, birthdays, family stats, relationship finder, and large-tree demo review.

### Public-Ready Architecture

Designed to evolve into a broader create-and-share product, while the birthday release keeps private data behind Firebase Auth and Firestore/Storage rules.

## Setup & Installation

1. Clone the repository:

   ```bash
   git clone <repo-link>
   ```

2. Open the project in VS Code.

3. Add your Firebase configuration in:

   ```text
   /js/firebase.js
   ```

4. Install dependencies:

   ```bash
   npm install
   ```

5. Use any local static server with Firebase-style route rewrites, or deploy to Firebase Hosting.

6. Add a sample person to ensure Auth, Firestore, and Storage connections are working.

## Local Checks

Install Node.js 20 or newer, then run:

```bash
npm run check
```

This checks JavaScript syntax, validates JSON config files, and scans for unsafe browser rendering APIs such as `innerHTML`.
